TXT에서 HTML 강의록 뷰어까지

수업 자료를 정적 웹 강의안으로 바꿉니다.

`txt-to-html-lecture-notes` 스킬은 원고처럼 들어온 TXT 자료를 페이지 데이터로 정리하고, 목차, 검색, 썸네일, 페이지 이동이 있는 HTML 뷰어로 출력하도록 에이전트에게 작업 방식을 제공합니다.

목차 검색 보기

문서 작성 과정을 영상처럼 따라갑니다.

아래 데모는 에이전트가 TXT를 읽고, 강의 구조를 잡고, HTML 뷰어 페이지를 만드는 과정을 자동 재생합니다. 실제 변환 결과물은 한 개의 정적 `index.html`로 배포할 수 있습니다.

01
TXT 입력수업 원고를 그대로 넣습니다.
02
구조 분석제목, 개념, 예시, 정리를 찾습니다.
03
페이지 데이터강의록 JSON 구조로 나눕니다.
04
HTML 뷰어목차와 검색이 있는 정적 페이지로 만듭니다.

NPM 명령으로 에이전트에 설치합니다.

      

        Skills CLI가 지원하는 글로벌 에이전트 대상에 `txt-to-html-lecture-notes` 스킬을 설치합니다.
        Codex, Claude Code, Gemini CLI, Antigravity 등 환경에 설치된 에이전트가 이 스킬을 사용할 수 있습니다.
      


      

        

          전체 지원 대상 설치
          npx skills add https://github.com/ai-studying-man/HTML_BOOK.git --skill txt-to-html-lecture-notes --agent "*" -g -y
        

        

          설치 확인
          npx skills list -g --json
        

      


      

        

          
정적 배포

          
빌드 도구 없이 HTML 파일만으로 GitHub Pages와 일반 정적 호스팅에 배포합니다.

        

        

          
학습자 중심 구조

          
TXT를 개요, 핵심 개념, 강의 내용, 예시, 정리로 재구성합니다.

        

        

          
뷰어형 경험

          
목차, 썸네일, 검색, 페이지 번호, 해시 링크를 통해 긴 강의록을 탐색합니다.
